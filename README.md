# Minecraft-JSON-Generator
Petit script fait rapidement, très très utile pour réaliser rapidement un texture pack pour sont serveur.

## Features
    • Write texts lang.
    • Write all langs.
    • Write attachables.
    • Write item.
    • Write item_texture.json.
    • Write terrain_texture.json.
    • Write blocks.json.
    
- [x] MCPE.
- [x] Fast.
- [x] Easy.
